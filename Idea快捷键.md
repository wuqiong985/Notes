#### 1.Idea快捷键


| 作用                  | 快捷键              |
| --------              | --------           |
| 生成Junit测试类文件  | Shift + Ctrl + T   |
| 快速生成Get Set方法  | Alt + Insert       |   
| 移动一整行           | Ctrl + Shift + ↑/↓ |   
| 替换字符串           | Ctrl+R             |
| 在全局中替换字符串      |Ctrl+Shift+R       |
| 查找当前变量的使用，并列表显示|	Alt+F7|
